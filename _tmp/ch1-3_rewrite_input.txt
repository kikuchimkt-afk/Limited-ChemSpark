Q  : イオン結合について、正しい記述を選びなさい。
   c1: ch1_3_q001_c1 :: 多数の金属原子が自由電子を介して結びつく結合である。
   c2: ch1_3_q001_c2 :: 分子と分子の間にはたらく弱い引力による結合である。
  ★c3: ch1_3_q001_c3 :: 陽イオンと陰イオンが静電気力（クーロン力）によって引き合ってできる結合である。
   c4: ch1_3_q001_c4 :: 二つの原子が互いに電子を出し合い、共有してできる結合である。
EXPL (current tts_explanation): イオン結合は、金属元素の原子が電子を放出して陽イオンに、非金属元素の原子が電子を受け取って陰イオンになり、これらが静電気力で引き合ってできる結合です。塩化ナトリウムがその代表例で、ナトリウムイオンと塩化物イオンが交互に規則正しく並んだ結晶をつくります。選択肢四は共有結合、三は金属結合、四は分子間力であり、いずれもイオン結合とは異なる概念です。

ID : ch1_3_q005
Q  : 電気陰性度について、正しい記述を選びなさい。
   c1: ch1_3_q005_c1 :: 原子が電子を放出して陽イオンになるのに必要なエネルギーである。
   c2: ch1_3_q005_c2 :: 共有結合において原子が共有電子対を引きつける強さで、ナトリウムが最大である。
   c3: ch1_3_q005_c3 :: 原子が電子を受け取って陰イオンになるときに放出するエネルギーである。
  ★c4: ch1_3_q005_c4 :: 共有結合において原子が共有電子対を引きつける強さの度合いを表した数値で、フッ素が最大である。
EXPL (current tts_explanation): 電気陰性度は共有結合中で原子が共有電子対をどれだけ強く引きつけるかを数値化したものです。周期表の右上に位置する元素ほど大きく、フッ素が最大値をとります。左下に位置する金属元素ほど小さくなります。貴ガスはほとんど化合物をつくらないため通常は電気陰性度を考えません。選択肢一はイオン化エネルギー、三は電子親和力との混同です。

Q  : 塩化ナトリウムの性質について、誤りを含む記述を選びなさい。
   c1: ch1_3_q017_c1 :: 融点は約八百一度と高い。
  ★c2: ch1_3_q017_c2 :: 固体でも液体でも電気を通さない。
   c3: ch1_3_q017_c3 :: 水に溶けやすい。
   c4: ch1_3_q017_c4 :: 無色あるいは白色の結晶である。
EXPL (current tts_explanation): 塩化ナトリウムは無色（白色）の結晶で水に溶けやすく、融点は約八百一度と高いイオン結晶です。固体状態ではイオンが固定されているため電気を通しませんが、融解状態や水溶液ではイオンが自由に動けるため電気を通します。選択肢二は『液体でも通さない』の部分が誤りです。

ID : ch1_3_q038
Q  : 金属結晶とイオン結晶の性質に関する記述のうち、誤りを含むものを選びなさい。
   c1: ch1_3_q038_c1 :: イオン結晶は力を加えると配列がずれて同符号イオンが反発し割れる。
   c2: ch1_3_q038_c2 :: 金属結晶は電気をよく通し、イオン結晶は固体で通さない。
  ★c3: ch1_3_q038_c3 :: イオン結晶は自由電子のおかげで展性と延性に富み、変形しやすい。
   c4: ch1_3_q038_c4 :: 金属結晶は自由電子がクッションの役割をするため展性がある。
EXPL (current tts_explanation): 展性（叩くと薄く広がる性質）と延性（引くと伸びる性質）は金属結晶の特徴であり、自由電子が配列のずれを許容するために生じます。イオン結晶には自由電子がなく、力で配列がずれると同じ電荷のイオンどうしが反発して割れてしまいます。選択肢三はイオン結晶と金属結晶の性質を完全に取り違えた誤りです。
